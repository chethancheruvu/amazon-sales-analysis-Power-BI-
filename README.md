# Amazon Sales Dashboard - Power BI

## Overview

This project presents an interactive Amazon Sales Dashboard developed using Power BI. The dashboard provides valuable insights into sales performance, profit trends, product analysis, category-wise performance, and geographical distribution of profits.

The goal of this project is to transform raw sales data into meaningful business insights that support data-driven decision-making.
